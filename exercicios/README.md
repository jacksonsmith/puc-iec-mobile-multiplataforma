## Política de prazos

- Entregas até 23:59 do prazo (horário Brasília).
- Atrasos: -10%/dia, máx 3 dias. Após isso, zero.
- 1 (uma) prorrogação justificada por aluno, mediante e-mail antes do prazo.

## Formato de entrega

- **Fork + PR** neste repositório, editando a pasta `pratica/` do exercício (ver [`COMO_ENTREGAR.md`](../COMO_ENTREGAR.md)).
- O autograder (**J.A.R.V.I.S.** 🤖) roda a cada commit do PR e comenta a nota mínima automática.
- Critérios manuais (relatório, screencast, apresentação) são avaliados no Canvas.

## Critério eliminatório

Projeto Final é **obrigatório**, em grupo (3-4 pessoas), com apresentação ao vivo + demo. Grupo que não apresentar não pontua no PF.
