# Atividade 1 - ADR Arquitetural

Entrega da Atividade 1 da disciplina Arquitetura Mobile Multiplataforma.

- ADR principal: [ADR-0001-stack-mobile-telemedicina.md](./ADR-0001-stack-mobile-telemedicina.md)

O cenário escolhido é uma plataforma de telemedicina B2C/B2B2C com restrições de LGPD, dados sensíveis de saúde, videochamada, push, biometria e operação em baixa conectividade. A decisão compara cinco alternativas de stack mobile e justifica a escolha por React Native com módulos nativos para os caminhos críticos.
